Learning Management System


> A complete Learning Management System designed to manage educational activities efficiently.
> This system helps students, teachers, and administrators organize courses, lessons, assignments, and learning progress.

Features
--------

> Student Management

> - Add new students  
> - Update student information  
> - View student profiles  
> - Track learning progress  
> - Manage student records  

> Course Management

> - Create new courses  
> - Add and manage lessons  
> - Update course details  
> - Organize learning materials  
> - View available courses  

> Teacher Management

> - Add teachers  
> - Manage teacher information  
> - Assign courses to teachers  
> - Manage teaching activities  

> Assignment System

> - Create assignments  
> - Submit assignments  
> - Track assignment status  
> - Store assignment records  

> Learning Progress

> - Monitor student performance  
> - Track completed courses  
> - Maintain learning history  
> - Generate progress reports  

Technologies Used
-----------------

> - Python  
> - Object-Oriented Programming (OOP)  
> - File Handling / Database Management  
> - Data Structures
